# An accurate Web version of Flappy Bird
Flappy Bird is usually stuck on mobile devices, and even worse 32bit mobile devices, meaning its becoming harder to play Flappy Bird, but now theres a Flappy Bird port on the web
# how it was made
The port was made by Google's Antigravity IDE using Gemini 3 Pro, the model looked at the original game's APK file and managed to port everything, even the thing that gives Flappy Bird the sprites from the single Spritesheet
